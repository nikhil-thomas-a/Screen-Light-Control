# Lume — Screen Light Control

> A minimal night-time reading app that dims and warms your screen for comfortable low-light use.

**🌐 Live:** [screen-light-control--nikhilthomasa.replit.app](https://screen-light-control--nikhilthomasa.replit.app)

---

## What it does

Lume is a browser-based tool that overlays a warm, dimmable tint over your screen — useful for reading at night without changing your OS display settings permanently.

- Adjustable brightness
- Adjustable warmth / colour temperature
- No installation — works in any browser tab
